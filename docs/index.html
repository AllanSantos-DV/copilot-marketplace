// como publicar

Cinco passos, em ordem. O último é o único que "publica".

  1. 01

    vender

    Coloque o runtime já empacotado em plugins/<nome>/ (nunca edite esses arquivos à mão).

  2. 02

    versão

    Suba o version no plugins/<nome>/plugin.json — é ele que o copilot plugin update compara.

  3. 03

    manifesto

    Reflita nome, versão e descrição em .github/plugin/marketplace.json (a fonte de verdade da vitrine).

  4. 04

    gerar

    Rode node docs/build.mjs para reassar a vitrine a partir do manifesto.

  5. 05

    commit

    Faça o commit em main — o commit é a publicação. Sem Actions, sem release.
